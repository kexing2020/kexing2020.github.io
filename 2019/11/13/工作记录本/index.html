  • 连接服务器

    • 命令行ssh root@192.168.2.112

      • 提示ssh：connect to host 192.168.2.112 port22：Connection timed out

      • 判断服务器没有安装ssh

        • 在服务器安装ssh步骤如下：
          • 切换到root用户su root
          • 查看ssh是否安装rpm -qa | grep ssh
          • 没有显示图形则输入命令行进行安装yum install openssh-server,一路yes到底
          • 安装完成以后启动/关闭/重启/状态service sshd start/stop/restart/status
          • 检查sshd服务是否开机启动chkconfig --list sshd
          • 设置开机启动chkconfig sshd on

      • 在git Bash输入ssh root@192.168.2.112命令行再次连接服务器

        • 显示以下内容则已连接服务器

  • git push出错：[remote rejected] master -> master (n/a (unpacker error))

    • 原因：服务器端目录没有写权限
    • 解决方法：在服务器端输入明令行chown -R 777 目录
